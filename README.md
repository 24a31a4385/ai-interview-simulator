# 🤖 Smart AI Interview Trainer

Live Demo: (https://24a31a4385.github.io/ai-interview-simulator)

## 🚀 Features
- Role-based interview (Frontend, Backend, HR)
- Difficulty levels (Easy, Medium, Hard)
- Timer for each question
- Score per answer
- Total score tracking
- Clean dark UI

## 🛠 Tech Stack
- HTML
- CSS
- JavaScript

## 📌 How it Works
1. Select a role and difficulty
2. Click "Start Interview"
3. Answer the question in the given time
4. Click "Submit" to see score and feedback

## 🔥 Future Improvements
- AI-based answer evaluation
- Voice interview
- Resume-based questions
